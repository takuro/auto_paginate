Javascript で半自動的にページ送りを生成するテンプレート

作成者：石井 拓郎
初版作成日：2010/10/26

検証した環境
Internet Explorer 8
Firefox 3.6
Google Chrome 7

使い方
普通は PHP なんかのサーバ側プログラムで制御しないといけないページ送りを，Javascript で実現するもの．
js/user.js の一番上，index オブジェクトに，ファイル名とページの簡単なタイトルを記入するだけ．
ここに書かれた順番でページ送りされます．
記入したタイトルは，これも半自動的に作られる目次に使われます．
「Next」リンクと「Previous」リンクは，HTML で div タグなんかの適当なタグに， next クラスと previous クラスを付けてあげるだけ．

使用例
e-Learning 教材で使うために使ったので，それには使えそう．

おまけ
キーボードショートカットで，ページ遷移ができます．
ただし IE だと動きませんでした．
 - 次へ：->，>
 - 前へ：<-，<
